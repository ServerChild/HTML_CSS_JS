自分ツッコミくま / ナガノのくま(농담곰 / 담곰이)는 일본의 크리에이터 나가노(ナガノ) 작가의 캐릭터. 2014년 나가노 작가가 라인 스탬프를 만들 때 自分ツッコミ(스스로 태클을 거는 것)가 메인인 스탬프를 만들려고 했는데, 글씨의 옆에 그다지 눈에 띄지 않는 캐릭터를 넣으려고 하얀 곰을 그렸던 것이 시작이었다. 일본명은 自分ツッコミくま로써 직역하면 '스스로 츳코미하는 곰'인데, 츳코미라는 표현을 그대로 쓰기도 어렵고, 설명하기도 다소 복잡하다 보니 영문명 joke bear를 국내로 들여오면서 그대로 받아들여 농담곰이란 이름을 채택한 듯하다.
